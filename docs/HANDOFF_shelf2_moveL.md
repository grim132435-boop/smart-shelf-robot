# 2층 매대 캔 적치 moveL 핸드오프 (MP팀 전달용)

작성 2026-06-15. 출처 = 우리 검증 파이프라인(E0509 + RH-P12-RN, GraspGen→cuRobo→Isaac Sim).
2층 매대(천장 있음·개구 16cm) 캔 직립 적치를 충돌 없이 성공한 값. **MP팀 충돌 원인 진단 + 좌표/조인트 제공.**

## ★ 먼저 — 2층서 부딪히는 주원인

2층은 **천장이 있고 개구가 16cm뿐**인데 캔이 13.5cm라 캔 위 여유가 ~2.5cm뿐.
**캔을 위쪽(상단)에서 잡으면** 캔 위로 솟은 그리퍼 + RealSense 카메라가 천장에 걸림 → 충돌.
- **처방 1: 캔을 "중앙"에서 잡을 것** (상단 파지 금지). 그래야 그리퍼 최고점이 천장 아래로 들어옴.
- **처방 2: 진입 시 캔을 앞턱 위로 최소만 띄울 것** (많이 들면 캔 윗면이 천장 침범).
- **처방 3: RealSense 충돌 모델 포함**해서 플래닝할 것 (카메라가 천장 간섭의 실제 주범).

## 매대 2층 좌표 (우리 USD 실측, world z)

| 항목 | z |
|---|---|
| 2층 바닥판 윗면 | 0.950 |
| 2층 앞턱 윗면 | 0.960 |
| 2층 천장 아랫면 | 1.110 (개구 0.160) |
| 개구 x폭 | 0.088 ~ 0.412 |
| 개구 y(앞턱~뒷벽) | 0.383 ~ 0.623 |

캔: 높이 0.135 (half 0.0675), 반경 0.03. 중앙 파지 시 grip_z_offset ≈ 0 (TCP=캔중심).

## moveL 시작 자세 (매대 앞 pre-insert) — 핵심

**TCP(EE=gripper_rh_p12_rn_base) world pose:**
- 위치: `[0.25, 0.22, 1.032]` m  (x=슬롯중앙, y=앞턱서 충분히 앞, z=진입높이)
- 자세 quat `(w,x,y,z)`: `[-0.5, 0.5, 0.5, 0.5]`  (≡ -q 동일회전)
  - 의미: 그리퍼 상하축(X) = world +Z(위), approach축(Z) = world +Y(매대 안쪽). → 캔 직립 유지.

**조인트값 [j1..j6] deg (우리 로봇 IK 해, 매대앞):**
`[-2.6, 32.0, 93.5, 88.1, 92.1, -35.9]`

## moveL 동작 (시작 자세에서)

1. **진입**: +y 직선, y `0.22 → 0.50`, z `1.032` 유지. (캔바닥이 앞턱 0.96 위로 통과)
2. **하강**: −z 직선, z `1.032 → 1.020`. (캔바닥이 바닥판 0.95 위 0.003 안착)
3. 그리퍼 부분 open(~0.35rad, 매대 측벽 간섭 회피) → −y 후퇴.

**적치 안착 조인트 [j1..j6] deg:** `[27.2, 41.1, 80.6, 105.1, 67.1, -34.7]`

## 높이 계산식 (프레임/매대 좌표 다르면 이걸로 환산)

```
entry TCP z = 앞턱top + 0.005 + half(0.0675) + grip_z_offset(중앙파지=0)
            = 0.960 + 0.005 + 0.0675 = 1.0325
place TCP z = 바닥top + 0.003 + half(0.0675) + 0
            = 0.950 + 0.003 + 0.0675 = 1.0205
```

## 우리 검증 마진 (그리퍼+RealSense+캔 vs 매대, 전부 양수)

- 진입 +25.9mm / 하강 +24.4mm / 후퇴 +43.5mm. 캔 직립 안착 z=1.018.

## 캔 파지 위치 z (로봇 base 기준) — MP팀 요청 2026-06-15

기준: 우리 로봇 base z = 0.73(world), 책상 윗면 = base − 0.03 = 0.70(world), 캔 height 0.135(half 0.0675).

| 위치 | world z | **base 기준 z (= world − 0.73)** |
|---|---|---|
| **캔 파지점(책상 위 픽, 직립 캔 중심, ★중앙파지)** | 0.768 | **+0.038 m** |
| 2층 적치 진입 TCP | 1.032 | +0.302 m |
| 2층 적치 안착 TCP | 1.020 | +0.290 m |

- 파지점 z = 책상윗면(base−0.03) + half(0.0675) ≈ **base + 0.038**. ★**캔 중앙**을 잡아야 함(상단 파지 시 2층 천장 간섭).
- 누운 캔도 위에서 캔 중심(축방향)을 잡으면 base 기준 z ≈ base + 반경(0.03) − 0.03 = base 근처(책상면)에서 잡힘
  → 이후 직립 재배향. 직립 적치 z는 위와 동일.
- ★ 이 z는 책상-base 상대높이(우리 −0.03)에 의존. MP팀 책상/base 관계가 다르면 "책상윗면 + half"로 환산.

## ⚠ 프레임 주의

- 위 world 좌표는 **우리 로봇 base = [-0.25, -0.04, 0.73]** 기준. MP팀 base/매대 위치가 다르면
  **world 좌표 대신 "매대 상대 + 높이 계산식"으로 환산**하거나, base 변환을 적용할 것.
- 조인트값은 **동일 E0509 kinematics + 동일 마운트**일 때만 그대로 사용. (우리는 j5≥0/j4±180° 비대칭
  한계 URDF 사용 — 자세는 유효하나 팀 로봇 한계와 대조 권장.)
- 더 필요하면 전체 moveL 웨이포인트(보간 trajectory)·attach 캔 부피·충돌구체(카메라 포함) yml도 제공 가능.
